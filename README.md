# 💪 MaxMuscle

A mobile fitness application prototype developed with **React Native** and **Expo** as part of an academic project for the Analysis and Systems Development program.

The application demonstrates a complete mobile interface flow, including authentication screens, user registration and a home dashboard focused on fitness applications.

---

# 📖 About

MaxMuscle was created to practice mobile application development using React Native and Expo.

The project focuses on building modern user interfaces, navigation between screens and form handling, providing a realistic prototype of a fitness application.

Although the application does not include a backend service, it demonstrates the complete front-end navigation flow and mobile UI development concepts.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Jhonbrayaan/Tela_Login.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm start
```

or

```bash
npm run android

npm run ios

npm run web
```
